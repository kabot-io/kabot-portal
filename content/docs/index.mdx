## Start the ROS 2 Web bridge

Install rosbridge suite:

```bash
sudo apt update
sudo apt install ros-$ROS_DISTRO-rosbridge-suite
```

start rosbridge

```bash
ros2 launch rosbridge_server rosbridge_websocket_launch.xml
```

you should see that the connection in upper left corners goes from disconnected to connected, if so you can proceed.

## Drive the Turtle

Start the [turtlesim](https://wiki.ros.org/turtlesim) node

```bash
ros2 run turtlesim turtlesim_node
```

Use the joystick below to move the turtle.
